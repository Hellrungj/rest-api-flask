# rest-api-flask

Basic TODO list restful api using Flask

## Setup

### Install

* install python 3
```apt-get install python3```
* install virtualenv
```pip3 install virtualenv```
* create a virtualenv
```virtualenv venv```
* activate virtualenv
```source venv/bin/activate```
* install Flask
```pip3 install Flask```
* install flask_restful
```pip3 install flask_restful```

## Run Server
```python api.py```
